문제 : centernumber

문제 설명 : 단어 s의 가운데 글자를 반환하는 함수, solution을 만들어 보세요. 단어의 길이가 짝수라면 가운데 두글자를 반환하면 됩니다.

1. 이해
    - 알고있는것 : 
        - 입력값 s = 단어,
        - 글자수는 짝수, 홀수 가 있다. 
    - 제한사항 : s는 길이가 1 이상, 100이하인 스트링입니다.

2. 계획
    - 단어의 길이를 알아낸다.
    - 길이가 홀수인지 짝수인지 구분한다.
    - 짝수면 가운데 두 글자를 반환한다.
    - 홀수면 가운데 한 글자를 반환한다.
